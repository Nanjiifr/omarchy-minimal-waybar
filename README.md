## Install
Clone the repo with `git clone https://github.com/Nanjiifr/omarchy-minimal-waybar.git` (https) or `git clone git@github.com:Nanjiifr/omarchy-minimal-waybar.git` (ssh).
Switch to the directory : `cd omarchy-minimal-waybar`  
Then run :
```
chmod +x install.sh
chmod +x uninstall.sh
./install.sh
```

## Uninstall
If you are unhappy with the waybar config, you can always restore the previous configuration (stored in `~/.config/waybar.bak`) by running `./uninstall.sh`
